# 🚀 快速开始指南

## 第一步：安装依赖

```bash
cd /Users/fukuanma/pet-ledger
npm install
```

## 第二步：启动开发服务器

```bash
npm run dev
```

然后在浏览器中打开 `http://localhost:5173`

## 第三步：开始使用

1. **添加交易** - 点击"添加交易"按钮，填写表单
2. **查看统计** - 在"统计分析"页面查看数据分析
3. **管理数据** - 在"设置"页面导出/导入数据

## 可用命令

```bash
npm run dev       # 启动开发服务器
npm run build     # 构建生产版本
npm run preview   # 预览生产版本
npm run type-check # 检查 TypeScript 类型
```

## 项目特点

✅ **跨平台** - 手机和电脑都能用  
✅ **离线优先** - 无网络也能记录  
✅ **云同步** - 可选的 Supabase 同步  
✅ **数据统计** - 多维度分析  
✅ **现代 UI** - Naive UI 组件库
